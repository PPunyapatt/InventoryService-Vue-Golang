## Testing Exam
---
This repository is a test for an interview. It is a website that displays inventory codes as barcodes and allows users to list, add, or delete inventory codes.

This project is built using **Vue.js** for the frontend and **Golang** for the backend and and **PostgreSQL** for the database.

## Running with docker
To run the application with docker, we need the following commands:
```
docker compose up -d
```
